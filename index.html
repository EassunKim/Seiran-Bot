Seiran Bot

This page is a list of features and commands within Matthew Kim's project "Seiran Bot"

Features

  • Call and answer: will respond to certain phrases from users
  • Custom commands (see below)

Commands

  • !roll
    Rolls a random number between 1 and 100
    Usage: !roll [Max roll value]
  • !help
    Directs users to this webpage for more information
  • !rps
    Starts a game of rock, paper scissors between two users
    Usage: !rps @[user]
